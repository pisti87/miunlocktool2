DATA COLLECTION

We are starting to collect data for program improvements. 
What we collect:
CPU, GPU, Windows version, system language, deviceID (in program settings), logs (after successful request submission).

All the provided data is stored for 90 days and can be deleted if asked so.
We do not share this data and use it to help you fix your problem that you have when running the program.
